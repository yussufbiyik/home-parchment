A fragile parchment that recalls you home. Crumbles to dust after a single journey. Fits naturally within vanilla gameplay.

> [!TIP]
> Right click to use your parchment to go back home!

> [!IMPORTANT]  
> This mod requires [Spellbook](https://www.curseforge.com/hytale/mods/spellbook) to function properly!
